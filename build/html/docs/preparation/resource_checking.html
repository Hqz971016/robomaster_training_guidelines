5.2.1. 资源盘点¶

参加机器人比赛需要耗费大量人力、物力，首先应明确学校方面能基于给予的资源，详见 依托组织 。战队如有需要，可考虑通过招商获取外部资源。招商，本质上是将自己现有的资源拿出去置换外部的资源，是一种互利互惠的行为。成功的资源置换建立在双方需求都得到一定满足的基础上。因此，在决定招商前，需要明确内部需求，内部可用于置换的资源，人力安排及组委会支持：

  1. 内部需求盘点

    • 备赛资金是否充足？

    • 流动资金是否充足？

    • 研发物资及生产加工需求情况如何？

    • 是否有其他需求待满足？

  2. 内部资源盘点

    • 是否有人力投入招商工作？

    • 可满足潜在赞助商的哪些需求？如曝光，雇主品牌建设，科技属性赋权，企业社会责任，品牌生态等。

  3. 队内分工

    • 队伍是否统一认知到招商的必要性？

    • 招商如何协调其他小组的资源？

如果赛队需要从外部获取资源，且有人力进行招商工作，则可以联系获取组委会招商人员，获取相关资料，规划队内分工，进行后续准备工作。联系方式及资料获取详见 官方渠道介绍 。
